# Minesweeper
A simple Minesweeper game clone made in Windows Forms (C#/.NET).


## Screenshots
![minesweeper](https://github.com/user-attachments/assets/3b7b0a94-25c8-4224-9c2b-d9955cce813a)


## Installing (Windows)
1. Download the [latest release](https://github.com/AlexMalfr/Minesweeper/releases/)
2. Extract the zip file
3. Run the executable (.exe) file


## How to compile
### Prerequisites
- Visual Studio 2019
- .NET Framework 4.7.2

### Compiling
1. Open the solution file (.sln) with Visual Studio
2. Build the solution (Ctrl + Shift + B)

### Running
3. Run the executable (.exe) file in the `bin/Debug/` folder
